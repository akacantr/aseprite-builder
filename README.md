# Aseprite Builder🎨

**Aseprite Builder** is a "one-click" automated pipeline for Windows designed to fetch, compile, and package the latest version of **Aseprite** from source. This project handles the entire environment configuration, dependency management (Skia), and compilation (CMake/Ninja).

---

## ✨ Features

* **GitHub Actions Ready:** Build Aseprite in the cloud in ~25 minutes without installing any software on your PC.
* **Dynamic Metadata Fetching:** Automatically finds the latest `Source.zip` from the official Aseprite repository.
* **Automated Skia Management:** Fetches and extracts the correct Skia graphics engine binaries.
* **One-Click Installer:** Generates a professional Windows Installer (`.exe`) using Inno Setup for easy distribution and uninstallation.
* **Absolute Path Logic:** Optimized to avoid common CMake "Relative Path" errors during configuration.

---

## 🚀 Usage Instructions

### Method 1: Cloud Build (Recommended)
Build Aseprite directly on GitHub's servers and download the finished installer.

1.  **Fork** this repository to your own account.
2.  Go to the **Actions** tab in your forked repository.
3.  On the left sidebar, click on **"Build Aseprite"**.
4.  Click the **"Run workflow"** dropdown button and click the green **"Run workflow"** button.
5.  **Wait ~25 minutes.** When the build is finished (green checkmark), go to the **Releases** section on your repository's sidebar to download your `aseprite-setup.exe`.

### Method 2: Local Build
1.  Ensure you have **Visual Studio 2022** (Desktop C++ workload), **CMake**, **Ninja**, and **Inno Setup** installed.
2.  Download the `build_aseprite.bat` from this repository.
3.  Right-click the script and select **"Run as Administrator"**.
4.  The script will automate the download and compilation process in a local `workspace` folder.

## 🛠️ Troubleshooting

* **"Source not found" Error:** Usually caused by an interrupted download or ZIP structure change. The current script uses recursive searching to find `CMakeLists.txt` to solve this.
* **"API rate limit exceeded":** The GitHub Action now uses the built-in `GITHUB_TOKEN` to bypass these limits.
* **Ninja Build Fails:** Ensure you have enough disk space (at least 5GB) and that your antivirus is not blocking the compiler.
